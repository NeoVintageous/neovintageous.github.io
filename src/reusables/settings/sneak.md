### vintageous_enable_sneak

- Type: `boolean`
- Default: `false`

Enable sneak plugin.

### vintageous_sneak_use_ic_scs

- Type: `integer`
- Default: `0`

Case sensitivity mode for Sneak.

Set this value to `0` for always case-sensitive.

Set this value to `1` for based on `'ignorecase'` and `'smartcase'`.
